# Radar Metrics Dashboard (frontend)

React + TypeScript + Vite + Ant Design client for the trend monitoring API. By default the app reads metrics from `http://localhost:8000/api` (override with `VITE_API_BASE_URL`).

## Requirements
- Node.js 18+
- npm 10+

## Setup
```bash
npm install
```

## Development
```bash
npm run dev
```
The dev server listens on http://localhost:5173.

## Production build
```bash
npm run build
```

## Preview production bundle
```bash
npm run preview
```
